My personal portfolio website showcasing my background, projects, technical skills, and contact information.

This project was built as a simple and responsive portfolio website using HTML and CSS.

## 🚀 Features

- Responsive design
- About Me section
- Technical Skills
- Project Showcase
- Contact Information
- Clean and minimal UI

## 🛠️ Technologies

- HTML5
- CSS3

## 🎯 Purpose

This project was created to present my software engineering portfolio in a clean and accessible way while practicing modern HTML and CSS fundamentals.
